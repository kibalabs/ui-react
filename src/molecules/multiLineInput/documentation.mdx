# MultiLineInput

**MultiLineInput** is a `molecule` component for multi-line text input. Wraps a textarea with InputWrapper styling.

## Theming

### CSS Class: `.KibaMultiLineInput`

See InputWrapper documentation for detailed theming options.

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaMultiLineInput {
    > textarea {
      width: 100%;
      min-height: 100px;
      resize: vertical;
      border: none;
      font-family: inherit;
      font-size: inherit;
    }
  }
}
```
