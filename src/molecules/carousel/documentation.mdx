## Theming

### CSS Class: `.KibaCarousel`

Carousel is primarily structural. Navigation buttons use IconButton theming.

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaCarousel {
    position: relative;

    > .KibaCarouselSlides {
      display: flex;
      overflow: hidden;
    }

    > .KibaCarouselNavigation {
      position: absolute;
      top: 50%;
      transform: translateY(-50%);
    }
  }
}
```
