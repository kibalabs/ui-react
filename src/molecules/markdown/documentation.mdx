## Theming

### CSS Class: `.KibaMarkdown`

Markdown renders standard HTML elements which inherit Text styling.

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaMarkdown {
    h1, h2, h3, h4, h5, h6 {
      margin-bottom: var(--kiba-spacing);
    }

    p {
      margin-bottom: var(--kiba-spacing);
    }

    ul, ol {
      padding-left: 1.5em;
      margin-bottom: var(--kiba-spacing);
    }

    a {
      color: var(--kiba-color-brand-primary);
    }

    img {
      max-width: 100%;
    }
  }
}
```
