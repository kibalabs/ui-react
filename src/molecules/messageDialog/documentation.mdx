# MessageDialog

**MessageDialog** is a `molecule` component for confirmation dialogs. Displays a title, message, and confirm/cancel buttons.

## Theming

### CSS Class: `.KibaMessageDialog`

Inherits styling from Dialog component. See Dialog documentation for theming details.

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaMessageDialog {
    > .KibaMessageDialogTitle {
      font-size: var(--kiba-text-font-size-header3);
      margin-bottom: var(--kiba-spacing);
    }

    > .KibaMessageDialogMessage {
      margin-bottom: var(--kiba-spacing);
    }

    > .KibaMessageDialogButtons {
      display: flex;
      justify-content: flex-end;
      gap: var(--kiba-spacing);
    }
  }
}
```
