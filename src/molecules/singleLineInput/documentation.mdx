## Theming

### CSS Class: `.KibaSingleLineInput`

See InputWrapper documentation for detailed theming options.

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaSingleLineInput {
    > input {
      width: 100%;
      border: none;
      font-family: inherit;
      font-size: inherit;
      background: transparent;
    }

    &.error {
      border-color: var(--kiba-color-error);
    }

    &.success {
      border-color: var(--kiba-color-success);
    }
  }
}
```
