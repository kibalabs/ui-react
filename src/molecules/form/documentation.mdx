## Theming

### CSS Class: `.KibaForm`

Form is a structural wrapper. Styling is applied to child input components.

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaForm {
    width: 100%;

    &.isLoading {
      opacity: 0.7;
      pointer-events: none;
    }
  }
}
```
