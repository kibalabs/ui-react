### CSS Class: `.KibaProgressCounter`

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaProgressCounter {
    display: flex;
    align-items: center;
    gap: var(--kiba-spacing);

    > .KibaProgressCounterItem {
      width: 2rem;
      height: 2rem;
      border-radius: 50%;
      display: flex;
      align-items: center;
      justify-content: center;
      background-color: var(--kiba-color-background-dark10);

      &.completed {
        background-color: var(--kiba-color-brand-primary);
        color: var(--kiba-color-text-on-brand);
      }

      &.current {
        border: 2px solid var(--kiba-color-brand-primary);
      }
    }
  }
}
```
