## Theming

### CSS Class: `.KibaMarkdownText`

Inherits styling from Text component. See Text documentation for theming details.

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaMarkdownText {
    a {
      color: var(--kiba-color-brand-primary);
      text-decoration: underline;
    }

    strong {
      font-weight: var(--kiba-font-weight-bold);
    }

    em {
      font-style: italic;
    }
  }
}
```
