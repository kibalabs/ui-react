**InputFrame** is a `molecule` component for basic input styling. Provides the visual frame for input elements.

## Theming

### CSS Class: `.KibaInputFrame`

See SingleLineInput and InputWrapper for detailed theming options.

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaInputFrame {
    border: 1px solid var(--kiba-color-background-dark10);
    border-radius: var(--kiba-border-radius);
    padding: var(--kiba-padding);
    background-color: var(--kiba-color-background);

    &:focus-within {
      border-color: var(--kiba-color-brand-primary);
    }
  }
}
```
