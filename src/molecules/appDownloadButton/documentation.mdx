## Theming

### CSS Class: `.KibaAppDownloadButton`

### Variants

| Variant | Description |
|---------|-------------|
| `dark` | Dark background button |
| `dark-clear` | Dark outline button |
| `light` | Light background button |
| `light-clear` | Light outline button |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaAppDownloadButton {
    display: inline-block;
    border-radius: var(--kiba-border-radius);
  }
}
```
