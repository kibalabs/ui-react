## Theming

### CSS Class: `.KibaTabBar`

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaTabBar {
    display: flex;
    border-bottom: 1px solid var(--kiba-color-background-dark10);

    > .KibaTabBarItem {
      padding: var(--kiba-padding);
      cursor: pointer;
      border-bottom: 2px solid transparent;

      &.selected {
        border-bottom-color: var(--kiba-color-brand-primary);
        color: var(--kiba-color-brand-primary);
      }

      &:hover:not(.selected) {
        background-color: var(--kiba-color-background-dark05);
      }
    }

    &.isFullWidth > .KibaTabBarItem {
      flex: 1;
      text-align: center;
    }
  }
}
```
