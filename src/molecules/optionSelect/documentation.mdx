## Theming

### CSS Class: `.KibaOptionSelect`

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaOptionSelect {
    position: relative;

    > .KibaOptionSelectTrigger {
      border: 1px solid var(--kiba-color-background-dark10);
      border-radius: var(--kiba-border-radius);
      padding: var(--kiba-padding);
      cursor: pointer;
    }

    > .KibaOptionSelectDropdown {
      position: absolute;
      top: 100%;
      left: 0;
      right: 0;
      background-color: var(--kiba-color-background);
      border: 1px solid var(--kiba-color-background-dark10);
      border-radius: var(--kiba-border-radius);
      box-shadow: var(--kiba-box-shadow);
      z-index: 10;
    }
  }
}
```
