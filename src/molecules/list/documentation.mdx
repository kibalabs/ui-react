## Theming

### CSS Class: `.KibaList`

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaList {
    list-style: none;
    padding: 0;
    margin: 0;

    > .KibaListItem {
      padding: var(--kiba-padding);
      cursor: pointer;

      &.selected {
        background-color: var(--kiba-color-brand-primary-clear90);
      }

      &.disabled {
        opacity: 0.5;
        cursor: not-allowed;
      }
    }

    &.shouldShowDividers > .KibaListItem {
      border-bottom: 1px solid var(--kiba-color-background-dark10);
    }
  }
}
```
