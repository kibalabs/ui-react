# PrettyText

**PrettyText** is an `atom` component for styled text with inline formatting. Inherits Text styles with support for bold, italic, etc.

## Theming

### CSS Class: `.KibaPrettyText`

Inherits all styles from `.KibaText`. See Text documentation for theming details.

### Variants

| Variant | Description |
|---------|-------------|
| `default` | Body text size |
| `header1` | Largest header |
| `header2` | Second level header |
| `header3` | Third level header |
| `header4` | Fourth level header |
| `header5` | Smallest header |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaPrettyText {
    b, strong {
      font-weight: var(--kiba-font-weight-bold);
    }

    i, em {
      font-style: italic;
    }

    u {
      text-decoration: underline;
    }
  }
}
```
