### Customizing with CSS

```css
@layer kiba-theme {
  .KibaBulletText {
    font-family: var(--kiba-text-font-family);
    font-size: var(--kiba-text-font-size);
    color: var(--kiba-color-text);
    line-height: var(--kiba-line-height);
  }
}
```
