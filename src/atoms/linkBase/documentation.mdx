## Theming

### CSS Class: `.KibaLinkBase`

### Variants

| Variant | Description |
|---------|-------------|
| `default` | No styling |
| `translucent` | Subtle hover effect |
| `card` | Card-like appearance |
| `image` | For wrapping images |

### States

| Class | Description |
|-------|-------------|
| `.fullWidth` | Expands to container width |
| `.fullHeight` | Expands to container height |
| `.disabled` | Disabled state |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaLinkBase {
    text-decoration: none;
    cursor: pointer;

    &.disabled {
      opacity: 0.5;
      pointer-events: none;
    }

    &.card {
      display: block;
      padding: var(--kiba-padding);
      border-radius: var(--kiba-border-radius);
      background-color: var(--kiba-color-background);
    }
  }
}
```
