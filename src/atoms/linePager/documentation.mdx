## Theming

### CSS Class: `.KibaLinePager`

### States

| Class | Description |
|-------|-------------|
| `.active` | Active page indicator |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaLinePager {
    display: flex;
    gap: var(--kiba-spacing);

    > .LinePagerItem {
      width: 0.5rem;
      height: 0.5rem;
      border-radius: 50%;
      background-color: var(--kiba-color-background-dark25);

      &.active {
        background-color: var(--kiba-color-brand-primary);
      }
    }
  }
}
```
