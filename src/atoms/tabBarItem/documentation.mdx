# TabBarItem

This component is only intended to be used with [Molecules/TabBar](/?path=/docs/molecules-progresscounter).

### Theming

TabBarItem styling uses CSS custom properties for colors and states:

```scss
.KibaTabBarItem {
  color: var(--kiba-color-brand-primary-clear25);
  border-bottom: 2px solid transparent;
  &:hover {
    background-color: var(--kiba-color-brand-primary-clear90);
  }
  &.selected {
    color: var(--kiba-color-brand-primary);
    border-bottom-color: var(--kiba-color-brand-primary);
  }
}
```
