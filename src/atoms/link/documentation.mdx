## Theming

### CSS Class: `.KibaLink`

### States

| Class | Description |
|-------|-------------|
| `.disabled` | When link is not clickable |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaLink {
    color: var(--kiba-color-brand-primary);
    text-decoration: underline;

    &:hover {
      color: var(--kiba-color-brand-secondary);
    }

    &:focus-visible {
      outline: 2px solid var(--kiba-color-brand-primary);
      outline-offset: 2px;
    }

    &.disabled {
      color: var(--kiba-color-disabled-text);
      cursor: not-allowed;
      text-decoration: none;
    }
  }
}
```
