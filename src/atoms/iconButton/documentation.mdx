## Theming

### CSS Class: `.KibaIconButton`

### Variants

| Variant | Description |
|---------|-------------|
| `default` | Transparent with brand color |
| `primary` | Filled brand background |
| `secondary` | Brand border |
| `tertiary` | No border, subtle background on hover |
| `small` | Smaller size |
| `large` | Larger size |
| `passive` | Muted styling |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaIconButton {
    border-radius: var(--kiba-border-radius);
    color: var(--kiba-color-brand-primary);
    padding: var(--kiba-padding);

    &.primary {
      background-color: var(--kiba-color-brand-primary);
      color: var(--kiba-color-text-on-brand);
    }

    &:hover {
      background-color: var(--kiba-color-brand-primary-clear90);
    }
  }
}
```
