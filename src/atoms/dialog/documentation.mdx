## Theming

### CSS Class: `.KibaDialog`

### States

| Class | Description |
|-------|-------------|
| `.closed` | When dialog is not visible |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaDialog {
    --kiba-dialog-backdrop-color: rgba(0, 0, 0, 0.5);
    background-color: var(--kiba-dialog-backdrop-color);

    > .KibaDialogInner {
      background-color: var(--kiba-color-background);
      border-radius: var(--kiba-border-radius);
      box-shadow: 0 4px 20px rgba(0, 0, 0, 0.15);
      padding: var(--kiba-padding-wide2);
    }

    &.closed {
      display: none;
    }
  }
}
```
