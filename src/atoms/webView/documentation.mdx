## Theming

### CSS Class: `.KibaWebView`

This component embeds external content and has minimal visual theming. The loading spinner follows LoadingSpinner theming.

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaWebView {
    position: relative;
    width: 100%;
    height: 100%;

    > .web-view-iframe {
      position: absolute;
      top: 0;
      left: 0;
      width: 100%;
      height: 100%;
      border: none;
    }

    > .web-view-loading-wrapper {
      position: absolute;
      top: 50%;
      left: 50%;
      transform: translate(-50%, -50%);
    }
  }
}
```
