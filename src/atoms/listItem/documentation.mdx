# ListItem

This component is only intended to be used with [Molecules/List](/?path=/docs/molecules-list).

### Theming

ListItem styling uses CSS custom properties for interactive states:

```scss
.KibaListItem {
  background-color: transparent;
  &.clickable:hover {
    background-color: var(--kiba-color-brand-primary-clear90);
  }
  &.selected {
    background-color: var(--kiba-color-brand-primary-clear80);
  }
}
```

Variants include: `slim`.
