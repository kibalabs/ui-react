# TitledCollapsibleBox

**TitledCollapsibleBox** is an `atom` component for collapsible sections with a built-in title header. Use `StatefulTitledCollapsibleBox` for self-managed state.

## Theming

### CSS Class: `.KibaTitledCollapsibleBox`

Inherits styles from CollapsibleBox. See CollapsibleBox documentation for base theming.

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaTitledCollapsibleBox {
    border: 1px solid var(--kiba-color-background-dark10);
    border-radius: var(--kiba-border-radius);

    > .KibaTitledCollapsibleBoxHeader {
      display: flex;
      justify-content: space-between;
      align-items: center;
      padding: var(--kiba-padding);
      background-color: var(--kiba-color-background-dark05);
      cursor: pointer;
    }

    > .KibaTitledCollapsibleBoxContent {
      padding: var(--kiba-padding);
    }
  }
}
```
