## Theming

### CSS Class: `.KibaSelectableView`

### States

| Class | Description |
|-------|-------------|
| `.selected` | Item is selected |
| `.disabled` | Item is disabled |
| `.fullWidth` | Expands to container width |
| `.fullHeight` | Expands to container height |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaSelectableView {
    position: relative;
    cursor: pointer;
    border: 2px solid transparent;
    border-radius: var(--kiba-border-radius);

    &.selected {
      border-color: var(--kiba-color-brand-primary);
    }

    &.disabled {
      opacity: 0.5;
      cursor: not-allowed;
    }

    > .KibaSelectableView-overlay {
      position: absolute;
      top: 0;
      left: 0;
    }
  }
}
```
