## Theming

### CSS Class: `.KibaButton`

### Variants

| Variant | Description |
|---------|-------------|
| `default` | Transparent with brand color text |
| `primary` | Filled brand background |
| `secondary` | Brand border, transparent background |
| `small` | Smaller padding and font |
| `large` | Larger padding and font |
| `card` | With shadow, card-like appearance |

Combine variants: `variant="primary-large"` or `variant="card-primary"`

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaButton {
    background-color: transparent;
    border-radius: var(--kiba-border-radius);
    border-width: 2px;
    color: var(--kiba-color-brand-primary);
    font-weight: 600;
    padding: var(--kiba-padding) var(--kiba-padding-wide);

    &:hover {
      background-color: var(--kiba-color-brand-primary-clear90);
    }

    &.primary {
      background-color: var(--kiba-color-brand-primary);
      color: var(--kiba-color-text-on-brand);
    }

    &.disabled {
      background-color: var(--kiba-color-disabled-light50);
      color: var(--kiba-color-disabled-text);
      cursor: not-allowed;
    }
  }
}
```
