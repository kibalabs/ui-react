### Theming

Button styling is controlled through CSS custom properties and variant classes. Define your button variants in your theme SCSS file:

```scss
.KibaButton {
  // Default button styling
  > .KibaButtonFocusFixer {
    background-color: transparent;
    border-radius: var(--kiba-border-radius);
    color: var(--kiba-color-brand-primary);
  }

  // Primary variant
  &.primary > .KibaButtonFocusFixer {
    background-color: var(--kiba-color-brand-primary);
    color: var(--kiba-color-text-on-brand);
  }
}
```
