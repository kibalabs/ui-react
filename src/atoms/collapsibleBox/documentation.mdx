## Theming

### CSS Class: `.KibaCollapsibleBox`

### Variants

| Variant | Description |
|---------|-------------|
| `unpaddedHeader` | Header without padding |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaCollapsibleBox {
    border: 1px solid var(--kiba-color-background-dark10);
    border-radius: var(--kiba-border-radius);

    > .KibaCollapsibleBoxHeader {
      background-color: var(--kiba-color-background-dark05);
      cursor: pointer;
    }

    &:hover > .KibaCollapsibleBoxHeader {
      background-color: var(--kiba-color-background-dark10);
    }

    > .KibaCollapsibleBoxContent {
      padding: var(--kiba-padding);
    }
  }
}
```
