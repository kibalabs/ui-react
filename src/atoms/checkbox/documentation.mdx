## Theming

### CSS Class: `.KibaCheckbox`

### States

| Class | Description |
|-------|-------------|
| `.checked` | When checkbox is selected |
| `.disabled` | When checkbox is disabled |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaCheckbox {
    > .checkbox-inner {
      border: 1px solid var(--kiba-color-text);
      border-radius: var(--kiba-border-radius);
      color: var(--kiba-color-text);
    }

    &.checked > .checkbox-inner {
      background-color: var(--kiba-color-brand-primary);
      border-color: var(--kiba-color-brand-primary);
      color: var(--kiba-color-text-on-brand);
    }

    &.disabled > .checkbox-inner {
      border-color: var(--kiba-color-disabled-text);
      opacity: 0.5;
    }

    > .checkbox-text {
      color: var(--kiba-color-text);
    }
  }
}
```
