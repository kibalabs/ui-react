### States

| Class | Description |
|-------|-------------|
| `.checked` | When switch is on |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaSwitch {
    > .switch-track {
      background-color: var(--kiba-color-background-dark25);
      border-radius: 1em;
      width: 3em;
      height: 1.5em;
    }

    > .switch-thumb {
      background-color: white;
      border-radius: 50%;
    }

    &.checked > .switch-track {
      background-color: var(--kiba-color-brand-primary);
    }
  }
}
```
