# KibaIcon

**KibaIcon** is a convenience wrapper around `Icon` that fetches icons from Kiba's CDN by ID. Supports multiple popular icon libraries.

## Theming

### CSS Class: `.KibaKibaIcon`

Inherits all styling from Icon. See Icon documentation for variants and customization.

### Supported Icon Sets

| Prefix | Icon Set | Example |
|--------|----------|---------|
| `ion-` | [Ionicons v5](https://ionic.io/ionicons) | `ion-home` |
| `mui-` | [Material Icons](https://fonts.google.com/icons) | `mui-home` |
| `bs-` | [Bootstrap Icons](https://icons.getbootstrap.com/) | `bs-house` |
| `feather-` | [Feather Icons](https://feathericons.com/) | `feather-home` |
| `remix-` | [Remix Icons](https://remixicon.com/) | `remix-home-line` |

### Note

Icons are fetched asynchronously from `assets-cdn.kiba.dev`. Consider using `Icon` with bundled SVGs for critical icons to avoid network delays.
