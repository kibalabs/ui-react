## Theming

### CSS Class: `.KibaLoadingSpinner`

### Variants

| Variant | Description |
|---------|-------------|
| `light` | White spinner |
| `dark` | Black spinner |
| `small` | 1rem size |
| (default) | 2rem size |
| `large` | 4rem size |
| `extraLarge` | 8rem size |
| `fill` | 100% of container |

Combine variants: `variant="light-large"`

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaLoadingSpinner {
    --kiba-loading-spinner-size: 2rem;
    --kiba-loading-spinner-width: 0.2em;
    --kiba-loading-spinner-color: var(--kiba-color-brand-primary);

    &.small {
      --kiba-loading-spinner-size: 1rem;
    }

    &.large {
      --kiba-loading-spinner-size: 4rem;
    }

    &.light {
      --kiba-loading-spinner-color: white;
    }

    &.dark {
      --kiba-loading-spinner-color: black;
    }
  }
}
```
