# Icon

**Icon** is a `particle` for displaying inline SVG icons. It renders raw SVG content and provides styling controls for color and size.

## Theming

### CSS Class: `.KibaIcon`

### Variants

| Variant | Size |
|---------|------|
| `small` | 0.75em |
| (default) | 1.3em |
| `large` | 2em |
| `extraLarge` | 4em |
| `extraExtraLarge` | 8em |
| `fillSize` | 100% (fills container) |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaIcon {
    width: var(--kiba-icon-size, 1.3em);
    height: var(--kiba-icon-size, 1.3em);

    &.fill > svg {
      fill: var(--kiba-icon-color, currentColor);
    }

    &.stroke > svg {
      stroke: var(--kiba-icon-color, currentColor);
    }

    &.small {
      --kiba-icon-size: 0.75em;
    }

    &.large {
      --kiba-icon-size: 2em;
    }
  }
}
```
