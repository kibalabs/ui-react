## Theming

### CSS Class: `.KibaImage`

### Variants

| Variant | Description |
|---------|-------------|
| `circular` / `profile` | Circular border-radius (50%) |
| `unrounded` | No border-radius |

### Fit Types

| Value | Description |
|-------|-------------|
| `cover` | Fills container, may crop |
| `contain` | Fits inside container, may letterbox |
| `crop` | Same as cover |
| `scale` | Fills container, may stretch |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaImage {
    border-radius: var(--kiba-border-radius);

    &.lazyloaded {
      transition: opacity 0.15s;
    }

    &.circular, &.profile {
      border-radius: 50%;
    }

    &.unrounded {
      border-radius: 0;
    }
  }
}
```

### Lazy Loading

Set `isLazyLoadable={true}` for lazy loading. Requires the [lazysizes](https://github.com/aFarkas/lazysizes) library. The component adds `.lazyload` / `.lazyloaded` classes automatically.
