## Theming

### CSS Class: `.KibaBox`

### Variants

| Variant | Description |
|---------|-------------|
| `card` | Elevated card with background, border, shadow, and padding |
| `bordered` | Box with border and padding |
| `padded` | Adds default padding |
| `transparent` | Transparent background with padding |
| `rounded` | Circular border-radius (50%) |
| `focusable` | Adds focus border styling |
| `unpadded` | Removes padding |
| `unmargined` | Removes margin |
| `overlay` | Semi-transparent background overlay |

### Customizing with SCSS

```css
@layer kiba-theme {
  .KibaBox {
    border-radius: var(--kiba-border-radius);

    &.card {
      background-color: var(--kiba-color-background-light10);
      border-color: var(--kiba-color-background-dark05);
      box-shadow: 0px 8px 8px -6px rgba(0, 0, 0, 0.15);
      padding: var(--kiba-padding-wide2);
    }

    &.bordered {
      background-color: var(--kiba-color-background-light10);
      border-color: var(--kiba-color-background-dark05);
      padding: var(--kiba-padding-wide2);
    }
  }
}
```

### Responsive Sizing

Box supports responsive dimensions via CSS custom properties with breakpoint suffixes:

```tsx
<Box
  width="100%"
  widthResponsive={{ small: '100%', medium: '50%', large: '33%' }}
/>
```

This sets `--kiba-box-width-sm`, `--kiba-box-width-md`, `--kiba-box-width-lg` which are used at their respective breakpoints.
