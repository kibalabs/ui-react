## Theming

### CSS Class: `.Divider`

### Variants

| Variant | Description |
|---------|-------------|
| `horizontal` | Full-width horizontal line (default) |
| `vertical` | Full-height vertical line |

### Customizing with CSS

```css
@layer kiba-theme {
  .Divider {
    color: var(--kiba-color-background-dark05);

    &.horizontal {
      border-bottom-width: 1px;
    }

    &.vertical {
      border-right-width: 1px;
    }

    &.strong {
      color: var(--kiba-color-background-dark25);
      border-width: 2px;
    }
  }
}
```
