## Theming

### CSS Class: `.KibaText`

### Variants

| Variant | Description |
|---------|-------------|
| `paragraph` | Block-level paragraph text (`<p>`) |
| `inline` | Inline text (`<span>`) |
| `bold` / `strong` | Bold weight text |
| `italic` / `emphasis` | Italic style text |
| `underline` / `inserted` | Underlined text |
| `deleted` | Strikethrough text |
| `mark` | Highlighted/marked text |
| `small` / `smaller` | Smaller font size |
| `large` | Larger font size |
| `subscript` | Subscript text |
| `superscript` | Superscript text |
| `header1` / `header` | H1 heading (3.0rem, 800 weight) |
| `header2` / `title` | H2 heading (2.0rem, 700 weight) |
| `header3` / `subtitle` | H3 heading (1.4rem, 700 weight) |
| `header4` | H4 heading (1.3rem, 600 weight, underlined) |
| `header5` | H5 heading (1.2rem, 600 weight) |
| `header6` | H6 heading (1.1rem, 600 weight) |
| `note` | Small, light text for annotations |
| `supersize` | Extra large text (3rem) |
| `colored` / `branded` | Brand primary color |
| `error` | Error color |
| `success` | Success color |
| `unmargined` | Removes margin |
| `margined` | Adds vertical margin |
| `inherit` | Inherits all styles from parent |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaText {
    font-family: var(--kiba-font-family);
    color: var(--kiba-color-text);
    line-height: var(--kiba-line-height);

    &.header1 {
      font-size: 3.0rem;
      font-weight: 800;
      line-height: 1.2em;
    }

    &.note {
      font-size: 0.8rem;
      color: var(--kiba-color-text-light25);
    }

    &.colored {
      color: var(--kiba-color-brand-primary);
    }
  }
}
```
