## Theming

### CSS Class: `.KibaVideo`

This component renders a native `<video>` element with minimal visual theming. Sizing is controlled via CSS custom properties.

### Fit Types

| Value | Description |
|-------|-------------|
| `cover` | Fills container, may crop |
| `contain` | Fits inside container, may letterbox |
| `crop` | Same as cover |
| `scale` | Fills container, may stretch |

### Customizing with CSS

```css
@layer kiba-structure {
  .KibaVideo {
    display: block;
    width: var(--kiba-video-width, auto);
    height: var(--kiba-video-height, auto);
    max-width: var(--kiba-video-max-width, none);
    max-height: var(--kiba-video-max-height, none);
    object-fit: var(--kiba-video-fit, fill);

    &.centered {
      margin-left: auto;
      margin-right: auto;
    }
  }
}
```
