## Theming

### CSS Class: `.KibaSpacing`

### Variants (PaddingSize)

| Variant | Description |
|---------|-------------|
| `none` | No spacing (0px) |
| `narrow2` | Extra narrow spacing |
| `narrow1` | Narrow spacing |
| `default` | Default spacing |
| `wide1` | Wide spacing |
| `wide2` | Extra wide spacing |
| `wide3` | Very wide spacing |
| `wide4` | Maximum wide spacing |

### Direction

| Value | Description |
|-------|-------------|
| `Both` | Adds margin on both axes (default) |
| `Horizontal` | Only horizontal (left) margin |
| `Vertical` | Only vertical (top) margin |

### Customizing with CSS

```css
:root {
  --kiba-padding: 0.5rem;
  --kiba-padding-narrow: 0.25rem;
  --kiba-padding-narrow2: 0.125rem;
  --kiba-padding-wide: 1rem;
  --kiba-padding-wide2: 1.5rem;
  --kiba-padding-wide3: 2rem;
  --kiba-padding-wide4: 3rem;
}
```
