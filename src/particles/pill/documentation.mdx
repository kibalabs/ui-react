## Theming

### CSS Class: `.KibaPill`

### Variants

| Variant | Description |
|---------|-------------|
| `default` | Light brand background |
| `primary` | Brand primary border |
| `secondary` | Filled brand background |
| `success` | Green success styling |
| `error` | Red error styling |

### Customizing with CSS

```css
@layer kiba-theme {
  .KibaPill {
    padding: var(--kiba-padding) var(--kiba-padding-wide2);
    border-radius: var(--kiba-padding-wide2);
    background-color: var(--kiba-color-brand-primary-clear75);
    color: var(--kiba-color-brand-primary);

    &.primary {
      border-color: var(--kiba-color-brand-primary);
    }

    &.secondary {
      background-color: var(--kiba-color-brand-primary-clear25);
      color: var(--kiba-color-text-on-brand);
    }

    &.error {
      background-color: var(--kiba-color-error-clear75);
      color: var(--kiba-color-error);
    }

    &.success {
      background-color: var(--kiba-color-success-clear75);
      color: var(--kiba-color-success);
    }
  }
}
```
